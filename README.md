# macbook_memo

## 快捷鍵

- 複製： command + c
- 貼上： command + v
- home: command + <-
- end:  command + ->
- 

## ssh to raspberry-pi

- find: `ping raspberrypi.local`
- 連線： `ssh pi@192.168.8.111`
- 

## vnc to rpi

- https://www.realvnc.com/en/connect/download/viewer/macos/
